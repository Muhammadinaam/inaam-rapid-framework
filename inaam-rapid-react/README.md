>

## Install

```bash
npm install --save inaam-rapid-react
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'inaam-rapid-react'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
